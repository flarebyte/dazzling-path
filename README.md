> Experimental module to deal with svg path


## Install

```sh
$ npm install --save dazzling-path
```


## Usage

```js
var dazzling-path = require('dazzling-path');

dazzling-path('Rainbow');
```

```sh
$ npm install --global dazzling-path
$ dazzling-path --help
```
